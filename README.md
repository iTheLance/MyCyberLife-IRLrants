## What this is

This is my personal technical writing archive. Everything here comes from real engagements, competitions, and late-night rabbit holes — not textbook theory. No fluff, no PR spin. Just how things actually played out.

You'll find:

- **CTF Write-ups** — Attack-defense, recon, API hunting, and token abuse from real competitions.
- **Binary & Malware Analysis** — Static and dynamic analysis of real-world applications.
- **Infrastructure & DevOps** — Container orchestration, cloud-native patterns, and the "why does this actually work?" deep dives.
- **Web App Pentesting** — Vulnerabilities, bypass techniques, and real case studies.
- **Whatever else I'm digging into** — If it's interesting enough to write about, it ends up here.
